🎨 Animated Hero Component Demo

📚 Integration Guide

1. Include the script:

<script src="assets/js/animated-hero.js"></script>

2. Create a container:

<div id="my-hero"></div>

3. Initialize in your JavaScript:

const hero = new AnimatedHero({ container: '#my-hero', titles: ['amazing', 'new', 'wonderful'], duration: 2000 // milliseconds }); hero.init(); // Add button callbacks (optional) hero.attachEventListeners({ onLaunchClick: () => console.log('Launch clicked'), onCallClick: () => console.log('Call clicked'), onSignupClick: () => console.log('Signup clicked') });

4. Available Methods:

hero.pause() // Pause title rotation hero.resume() // Resume title rotation hero.destroy() // Cleanup component

5. Dependencies:

✅ design-system.css (for styling) ✅ Font Awesome (for icons via <i> tags) ✅ No external libraries needed (pure vanilla JS)
